APP Class Reference HomePage

The following sections describe the APP classes. They are concerned of app. We could only use them in app project.

2.1 PublicLogger

prototype:

class PublicLogger : public Logger

head file:

app_logs.h

description:

A derived class from class Logger. It write log messages to public log. The path is "/var/log/app_public.log" by default. We defined a singleton object of it. We could use PublicLoggerInstance::instance() to reference the singleton object.
Please consult class LoggerCtrl about where to use this singleton.
